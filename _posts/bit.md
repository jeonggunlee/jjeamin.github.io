# 비트코인 예측기

```
Input -> Model -> Output  
```

# Input

## Core Feature

- **30틱 기준**
- 시가, 종가, 고가, 저가 => `지지선`,`저항선`
- `지지선`,`저항선`을 기준으로 라벨링
- `지지선`에 가까울수록 `1`에 가까운 값
- `저항선`에 가까울수록 `0`에 가까운 값

## ++ Feature

- 추가적인 Feature

# Model

- LSTM, GRU, CNN etc.

# Output
- 오르는지 내리는지에 대한 확률 지표
- `0 ~ 1`
- `음수` : 내려갈 확률
- `양수` : 올라갈 확률


## 시나리오

```
Input : 오늘 포함 이전 30 틱 데이터(시가, 종가, 고가, 저가, 지지, 저항, 추가 Feature)

Model

Ouput : 0 ~ 1사이 확률 값
```
